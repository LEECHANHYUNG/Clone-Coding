#사용자 지정 변수<br>
- 사용자 지정 변수는 사용자가 직접 정의하는 변수로 재사용성이 높다.<br>

1) 사용자 지정 변수의 상속<br>
- 사용자가 원하는 클래스에만 해당 변수가 사용이 가능하도록 선언가능.<br>

2) calc css함수<br>
- 사용자 지정 변수를 정수값으로 정의하고 사용할 때는 calc함수를 통해 사용이 가능<br>

3) 대체변수 정의<br>
- var()안에 첫 번째 변수가 아직 정의되지 않은 경우 여러 개의 대체변수 지정 가능<br>
